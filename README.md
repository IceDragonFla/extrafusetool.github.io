# 拼豆辅助工具 (Fuse Bead Tool)

一个基于浏览器的拼豆图案辅助工具：上传图片后可自动量化到拼豆色号，并提供颜色聚焦、替换、套盒筛选、切片制作等能力，帮助你更快完成实际摆豆。

## 项目亮点

- 本地处理：图像处理全程在浏览器端完成，不上传图片
- 多品牌支持：覆盖 Mard / 咪小窝 / 黄豆豆 / coco / 漫漫 / 小舞 / 盼盼 / Hama / Perler / Artkal 等
- 颜色工作流完整：色号映射、缺货规避、颜色替换、颜色聚焦、统计排序
- 制作友好：支持图块切片、坐标标尺、镜像与像素块预处理

## 核心功能

### 1) 图像转拼豆
- 自动将像素映射到最接近的拼豆色号
- 可切换品牌调色板
- 支持上传 PNG / JPEG / GIF

### 2) 颜色管理
- 颜色聚焦条：单选/多选快速聚焦目标颜色
- 颜色替换：支持“全局替换”和“单像素替换”
- 缺货色号管理：将缺货色号排除在匹配结果外
- 功能色筛选（Mard）：夜光 / 果冻 / 珠光 / 温变

### 3) 套盒与排序
- 套盒模式：按套盒约束可用颜色
- 颜色统计支持多种排序：按数量 / 按色号 / 按套盒
- 套盒维度筛选显示

### 4) 视图与制作辅助
- 完整图、切片图、可拖拽放大视图
- 坐标标尺显示，方便定位摆放
- 水平/垂直镜像
- 像素块预处理（框选网格区域后压缩重建）

## 最近更新（v1.3.0）

### 新增
- 新增“少量色块自动合并”算法：自动处理数量极少、连通块很小的杂色点
- 为该算法新增独立开关，可按需启用/关闭，并自动记忆设置

### 修复
- 修复颜色聚焦条在重绘后无法继续聚焦的问题
- 统一颜色筛选键值逻辑，避免多次筛选后状态异常

### 优化
- 优化聚焦条事件绑定策略，避免重复绑定引发的交互不稳定

## 快速开始

1. 打开网页并上传图片
2. 选择拼豆品牌与图块大小
3. （可选）设置功能色、套盒模式、缺货色号
4. 在颜色聚焦条中筛选要优先摆放的颜色
5. 使用切片图与坐标辅助完成分块制作

## 使用建议

- 建议优先使用边缘清晰、分辨率适中的图片
- 如果出现零星杂色，保持“少量色块自动合并”开启
- 若需严格保留原始细节，可关闭该开关再重新量化

## 隐私说明

所有图形处理工作都在浏览器本地进行，不会上传或保存您的图片。
